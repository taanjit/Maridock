SECTION 2: DESCRIPTION

This specification covers the overhaul, inspection, calibration, and reinstallation of Main Engine Fuel Injection Pumps including shock absorbers for the [MAKE: _____ | MODEL: _____ | QUANTITY: _____ PUMPS | POWER: _____ PS | RPM: _____].

Location: Engine Room

System Type: Mechanical/Hydraulic

Criticality Level: High

SECTION 3: SCOPE OF WORK / INSTRUCTIONS
Preparatory Measures:
  • Obtain necessary permits and safety clearances
  • Isolate system and implement LOTO (Lock Out Tag Out) procedures
  • Prepare work area with adequate lighting, ventilation, and access
  • Brief all personnel on safety requirements and work procedures
  • Isolate fuel system, drain and disconnect fuel lines
  • Arrange for maker's authorized service engineer attendance
Job Breakdown:
  1. Removal and Documentation: Remove fuel pumps along with shock absorbers, tag each unit and document position, transport to workshop with proper handling procedures
  2. Dismantling and Inspection: Dismantle each unit completely, clean and inspect for wear, corrosion or damage, measure all relevant components including fuel pump leads and shim thickness
  3. Reconditioning and Calibration: Renew O-rings, seals, springs, and worn/damaged parts, reassemble as per maker's specifications, calibrate each pump and document timing settings
  4. Testing and Verification: Overhaul and function test shock absorbers, conduct function test on test bench if available, perform fuel leak tests and pressure checks, present to Class surveyor for CSM crediting
  5. Installation and Commissioning: Return pumps to vessel and refit in original positions, torque tighten mounting bolts, adjust and verify fuel injection timing, conduct operational checks during trials
Safety Controls:
  • All work to be performed under permit to work system
  • Use appropriate PPE: Safety helmets, safety shoes, coveralls, safety glasses, chemical-resistant gloves
  • Ensure proper ventilation and gas-free certification if required
  • Implement fall protection measures where applicable
  • Handle fuel systems with extreme care - no ignition sources
  • Proper fuel spill containment and cleanup procedures
  • MARPOL compliance for waste oil and fuel disposal
Standards & Compliance:
  • Work to be performed in accordance with manufacturer's instructions
  • Comply with class requirements: [CLASS_REQUIREMENTS]
  • Follow company safety management system procedures
  • Adhere to port state and flag state regulations
Deliverables:

  • Calibration certificates and timing documentation
  • Photographic documentation of work progress
  • Updated maintenance records with before/after measurements
  • Class survey certification for CSM crediting
